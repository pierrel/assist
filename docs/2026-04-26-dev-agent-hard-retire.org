The conversion:

#+begin_src python
self.workspace = tempfile.mkdtemp()
# Project-indicator so create_agent's general prompt pre-loads the dev skill body
# (matches the pattern in test_dev_skill_multi_turn.py).
with open(os.path.join(self.workspace, "pyproject.toml"), "w") as f:
    f.write('[project]\nname = "test-project"\n')
self.sandbox = SandboxManager.get_sandbox_backend(self.workspace)
if self.sandbox is None:
    self.skipTest("Docker sandbox unavailable")

# in _create_agent:
return AgentHarness(create_agent(
    self.model,
    working_dir=self.workspace,
    sandbox_backend=self.sandbox,
))
#+end_src

The =project_indicators= seed (=pyproject.toml=) is critical — without it, the general agent's prompt won't pre-load the dev skill body, and the small model is unreliable at exercising progressive disclosure on its own.  See the org-format-skill canary findings in [[file:2026-04-26-skills-system.org][skills-system.org]].
